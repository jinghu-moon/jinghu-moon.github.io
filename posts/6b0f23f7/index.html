博客首页添加公告栏

本文最后更新于：16 分钟前

本教程适用于 Fluid 主题，别的主题，按照主题代码结构，简单修改即可。

1.1 broadcast.ejs

进入 Blog\themes\fluid\layout_partial 路径，新建 broadcast.ejs 文件。添加以下代码。

<% if (theme.broadcast.enable && !page.layout){ %>
    <div class="notice">
        <svg class="icon-laba" aria-hidden="true">
            <use xlink:href="#icon-laba"></use>
        </svg>
        <span class="notice-content">
            <span class="element"></span>
        </span>
    </div>
    <script src="https://cdn.bootcdn.net/ajax/libs/typed.js/2.0.12/typed.js"></script>
    <script>
        var typed = new Typed(".element", {
            strings: ["<%= theme.broadcast.text %>"],  //输入内容, 支持html标签
            startDelay: 0,                             //开始延迟
            typeSpeed: 50,                             //打字速度，数字越大，速度越慢。
            backSpeed: 75,                             //回退速度
            loop: false,                               //是否循环
            showCursor: true,                          //显示游标
            cursorChar: '_',                           //游标样式
        });
    </script>
<% } %>

1.3 layout.ejs

进入 Blog\themes\fluid\layout 路径，打开 layout.ejs 文件。添加以下代码。

<main>
    <% if(is_post() || page.layout === '404') { %>
      <%- body %>
    <% } else { %>
      <div class="container nopadding-x-md">
        <!-- 公告栏 添加代码 -->
        <div id="board" class="notice_board">
          <% if (theme.broadcast.enable){ %>
          <%- partial('_partial/broadcast') %>
          <% } %>
        </div>

1.4 _config.fluid.yml

在 _config.fluid.yml 中添加以下代码。

# 公告栏
broadcast:
  enable: true   				  # true 开启，false 关闭。
  icon: "iconfont icon-laba"  	   # fontawesome 图标库
  text: 看的文章越多，越深感自己的不足，后面的时间，我将把精力放在学习新知识上。博客将停更一段时间。等我回来！

建议 text 字数不要超过 50。字数大于 63，就会换行。

引入 JS。

custom_js:
  - //at.alicdn.com/t/font_2976433_kdpcclj023.js

使用 阿里巴巴矢量图标库 的彩色图标，需要引入 JS，不是引入 CSS。

使用方法： 图标使用介绍 - 阿里巴巴矢量图标库 。

同时测试得：font-class 引用，不能实现图标动画效果。symbol 引用，可以实现。
