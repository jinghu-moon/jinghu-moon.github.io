微信通知 API 实现 Twikoo 新消息提醒

本文最后更新于：10 天前

本文教程适用于 Twikoo V1.4.5 及以上。V1.4.5 以下的同学，建议升级至最新版本或者看文末的参考文章

1.1 注册企业微信

企业微信 (qq.com) ，如下。

1.2 创建应用

注册成功后，点「管理企业」进入管理界面，选择「应用管理」 → 「自建」 → 「创建应用」。创建完成后进入应用详情页，得到 agentid 、secret ，保存下来。 Secret 在手机企业微信中查看。

手机端消息页面，查看 Secret。

网页端「我的企业」底部，保存 企业ID 。

1.3 创建 API 云函数

首先到 云开发 CloudBase - 腾讯云 创建一个环境，不会创建环境的朋友请看 Twikoo 腾讯云云开发部署视频教程 - bilibili 。当然也可以使用 Twikoo 的云开发环境。

创建一个云函数，可取名为 weixin-push 。选择 helloworld 空白模板函数，Python3.6 环境， 128MB 运行内存，点击下一步，创建云函数。粘贴下面的代码，点击创建。

进入云函数，函数代码界面，建一个以 .py 为后缀的文件，文件名任意。粘贴下面的代码，保存。

# -*- coding: utf8 -*-
import requests
import json

def getTocken(id,secert,msg,agentId):
    url = "https://qyapi.weixin.qq.com/cgi-bin/gettoken?corpid=" + id + "&corpsecret=" + secert

    r =requests.get(url)
    tocken_json = json.loads(r.text)
    # print(tocken_json['access_token'])
    sendText(tocken=tocken_json['access_token'],agentId=agentId,msg=msg)

def sendText(tocken,agentId,msg):
    sendUrl = "https://qyapi.weixin.qq.com/cgi-bin/message/send?access_token=" + tocken
    # print(sendUrl)
    data = json.dumps({
        "safe": 0,
        "touser" : "@all",
        "msgtype" : "text",
        "agentid" : agentId,
        "text" : {
            "content" : msg
        }
    })
    requests.post(sendUrl,data)

def main(event, context):
    try:
        apiid=event['queryStringParameters']['id']
        apisecert=event['queryStringParameters']['secert']
        apiagentId = event['queryStringParameters']['agentId']
        apimsg = event['queryStringParameters']['msg']
    except:
        apimsg = '有必填参数没有填写，请检查是否填写正确和格式是否错误。详情可以参阅：https://blog.zhheo.com/p/1e9f35bc.html'
        status = 1
    else:
        try:
            # 执行主程序
            getTocken(id=apiid,secert=apisecert,msg=apimsg,agentId=apiagentId)
        except:
            status = 1
            apimsg = '主程序运行时出现错误，请检查参数是否填写正确。详情可以参阅：https://blog.zhheo.com/p/1e9f35bc.html'
        else:
            status = 0
    # print(event)
    # print("Received event: " + json.dumps(event, indent = 2)) 
    # print("Received context: " + str(context))
    # print("Hello world")
    status_str = json.dumps({
            "status":status,
            "msg":apimsg
        })
    return(status_str)

1.4 配置访问服务

如果是第一次创建，参考 Twikoo 教程配置安全域名。

至此，云函数创建完成。

1.5 测试服务

使用以下方式，拼接成一个 URL，浏览器访问，看看手机微信 / 企业微信是否收到消息。

云函数的访问服务 URL + 触发路径 + ?id = 企业微信 ID + secert = 应用 secert + Id = 应用 AgentId + &msg =

空格和加号仅为显示清楚，URL 中没有空格、加号。

访问服务 URL：在「访问服务」界面，点击 weixin-push 。进入新页面后，地址栏里的网址。

URL 示例：

https://twikoo-xxxxxxxxxx-1305947611.ap-shanghai.app.tcloudbase.com/weixin-push?id=wwd2bxxxx499947417&secert=NLCixxxxQbDcyYstqxYD-9ww8OP9Uxxxxx9D4E&agentId=1000xxx&msg=hello

返回以下内容，则说明配置成功。

{"status": 0, "msg": "hello"}

将 URL 复制到 Twikoo 设置的对应位置。大功告成。

1.6 快速查看消息

打开手机企业微信，处于「消息」一栏，左滑，点击小齿轮，进入设置。「新消息通知」→「仅在企业微信中接受消息」，关闭。然后可以卸载企业微信。

打开微信，你会在「通讯录」看到「我的企业」一行，进入，将应用添加至桌面。
